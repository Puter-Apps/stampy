# Stampy

Chat with any websites, powered by [Puter.js](https://docs.puter.com)!

![](screenshot.jpg)

Add multiple websites and have conversation with accurate data with each websites.

## Technology

- Puter AI for LLM chat
- Puter KV store for persisting website list
- Puter Networking for retrieving HTML page
- Puter FS for storing search index and documents

With Puter.js, you gain access to these powerful APIs without needing to manage a server.
